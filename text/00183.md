## Express 104: Отображение информации

Веселый урок! Вы создадите несколько представлений (view) и соберете ваше приложение по кусочкам. Наконец-то мы увидим результат в браузере! Этот урок длинный и разбит на несколько вспомогательных уроков. Обязательно просмотрите все.

### По окончанию

К концу этого урока вы сможете:

- Рассказать о асинхронных функциях;
- Управлять асинхронными операциями в контроллерах;
- Управлять потоком выполнения при использовании асинхронных операций;
- Создавайте и расширяйте шаблоны;
- Писать шаблоны с помощью Pug;
- Передавать информацию в шаблон;
- Создавать страницы с правами только для чтения;
- Рассказать, как работают маршруты (routes);
- Рассказать, как view и model работают на практике;
- Создавать запросы в базу данных с помощью моделей;
- Управлять датами с помощью moment.
